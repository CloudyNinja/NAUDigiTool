Deciphering the Truth Table

    A Truth Table is an organized table showing outputs based on specific inputs.  The columns on the left of the dividing line are inputs.  The column(s) to the right of the dividing line are output(s). 

 

The inputs are put through some sort of logic network to produce the outputs.  This logic network can be represented by a boolean equation.  The boolean equation can be found by using a Karnaugh Map.
